css위치속성 

요소의 위치 지정 형식을 설정할 때는 position 속성을 사용한다.

1. absolute : 부모기준, 절대 위치 좌표 설정
2. static : 위쪽에서 아래쪽으로 순서대로 배치
3. relative : 초기 위치에서 상하좌우로 위치이동
4. fixed : 부모기준이 아닌 전체영역 기준, 절대 위치 좌표 설정
